## Quick Start

Follow these steps to set up your first documentation site.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://pareteknoloji.com` and verify your email.
  </Step>
  <Step title="New Space" icon="plus">
    Click "New Documentation Space" and name it "My Project Docs".
  </Step>
  <Step title="Upload Files" icon="upload">
    Connect your GitHub repo or drag-and-drop MDX files.
  </Step>
  <Step title="Publish" icon="globe">
    Set brand color to `#3B82F6` and hit "Publish".
  </Step>
</Steps>

## Brand Guidelines

Maintain consistency across your docs. Use the primary color `#3B82F6` for accents.

<Tabs>
  <Tab title="Colors" icon="palette">
    - Primary: `#3B82F6`
    - Secondary: `#1E40AF`
    - Background: `#FFFFFF`

    Customize in space settings:

    ```json
    {
      "brandColor": "#3B82F6",
      "logo": "https://your-logo.com/pare.svg"
    }
    ```
  </Tab>
  <Tab title="Typography" icon="font">
    Use system fonts for performance:

    ```css
    body {
      font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;
    }
    ```
  </Tab>
  <Tab title="Icons" icon="star">
    Leverage Lucide icons like `code`, `book-open`, and `zap`.
  </Tab>
</Tabs>

## Integration Examples

Embed dynamic content using our API.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.pareteknoloji.com/docs/YOUR_SPACE/search', {
    headers: { 'Authorization': `Bearer ${YOUR_API_KEY}` }
  });
  const docs = await response.json();
  console.log(docs);
  ```
  ```python
  import requests

  headers = {'Authorization': f'Bearer {YOUR_API_KEY}'}
  response = requests.get('https://api.pareteknoloji.com/docs/YOUR_SPACE/search', headers=headers)
  docs = response.json()
  print(docs)
  ```
</CodeGroup>
